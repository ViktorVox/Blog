# Персональный Блог

Привет! Это мой пет-проект - личный блог Fullstack разработчика.
Здесь я публикую свои мысли о разработке, технологиях и пути в IT.

Проект создан для демонстрации навыков работы с **Laravel**, проектирования архитектуры БД и создания веб-приложений.

## Технологический стек

* **Backend:** PHP 8.2, Laravel 11
* **Database:** MySQL (с использованием Migrations & Seeders)
* **Frontend:** Blade Templates, Tailwind CSS
* **Server:** Apache (XAMPP)

## Функционал

* **Аутентификация:** Вход и регистрация пользователей (Laravel Auth).
* **CRUD Постов:** Создание, чтение, удаление постов через админ-панель.
* **Безопасность:** Валидация данных (`$request->validate`), защита от CSRF.
* **UI/UX:** Адаптивная верстка на Tailwind CSS.
* **Разграничение прав:** Кнопка "Написать пост" доступна только авторизованным пользователям.

## Структура проекта

Проект следует классической MVC архитектуре Laravel:
* `app/Http/Controllers/PostController.php` — Логика работы с постами.
* `database/migrations/` — Структура базы данных (нормализованная).
* `resources/views/` — Шаблоны страниц (Blade).

## Как запустить локально

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/ViktorVox/Blog.git
    ```
2.  **Установите зависимости:**
    ```bash
    composer install
    npm install
    ```
3.  **Настройте окружение:**
    * Скопируйте `.env.example` в `.env`
    * Пропишите настройки БД (DB_DATABASE, DB_USERNAME и т.д.)
4.  **Запустите миграции:**
    ```bash
    php artisan migrate
    ```
5.  **Запустите сервер:**
    ```bash
    php artisan serve
    ```
